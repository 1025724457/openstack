OpenStack
=========

《OpenStack开源云王者归来》一书所附代码及资料。本书采用OpenStack Grizzly版本。

在使用资料包之前，应该选择一种方式：

- 创建链接文件，节省空间，但是不能改变目录树相对结构。
- 创建拷贝文件，不节省空间，但是每个章节可以单独提取出来使用，不需要依赖于整个资料包。
